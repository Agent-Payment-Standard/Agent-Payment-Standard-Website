# Agent Payment Standard Website

Official website for the **Agent Payment Standard (APS)**, built with [MkDocs Material](https://squidfunk.github.io/mkdocs-material/) and hosted via GitHub Pages.

## 🚀 Getting Started

This repository contains the source code for the APS documentation site.

- **Content**: Edit Markdown files in the `docs/` directory.
- **Theme/Config**: Modify `mkdocs.yml` to change the layout, colors, or navigation.
- **Deployment**: Any push to the `main` or `master` branch will trigger the GitHub Action in `.github/workflows/deploy.yml` to automatically deploy the site.

## 📁 Structure

- `docs/`: Source Markdown files and static assets.
- `mkdocs.yml`: Main configuration file.
- `.github/`: Automatic deployment workflow.
- `.gitignore`: Standard exclusions for Python/MkDocs build artifacts.
